# SimpleScheduler

## How to build
首先下载qt：https://www.qt.io/download
选择Qt6.4.0 勾选MSVC，其他都不需要。

将msvc2019_64\bin添加到环境变量的path里面

下载cmake与vs

https://cmake.org/files/v3.24/

https://visualstudio.microsoft.com/zh-hans/vs/

推荐使用vscode下载cmake插件开发
如果喜欢命令行可以用以下命令完成build
~~~
cmake -B build
cmake --build build
~~~
